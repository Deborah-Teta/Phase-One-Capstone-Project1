About Book Explorer

Our Mission

Book Explorer is designed to help readers discover and organize their reading interests. We provide easy access to a huge library of books through the Open Library API.

Features

  • ✓ Search through thousands of books by title
  • ✓ Save your favorite books for easy access
  • ✓ Responsive design for all devices
  • ✓ Persistent favorites using local storage

Technologies Used

Built with HTML, CSS (Tailwind), JavaScript, and the Open Library API. This project demonstrates modern web development practices including responsive design, async programming, and modular JavaScript architecture.
